Axioms of Set Theory

2.2 Functions

Definition 2.7 (Functions) A relation \(f\) is a function or mapping if \[ \forall x, y, z, (xfy \vee xfz \implies y = z) \] or that for every \(x \in \dom f\), there is a unique \(y\) with \(xfy\).

The unique \(y\) is defined as \(f(x)\) if \(x \in \dom f\), i.e., \((x, y) \in f \iff x \in \dom f \vee y = f(x)\).
Proposition 2.2 For all functions \(f, g\), \(f = g\) iff \(\dom f = \dom g\) and \(f(x) = g(x)\) for all \(x \in \dom f\).
Proof. If \(\dom f = \dom g\) and \(\forall x \in \dom f, (f(x) = g(x))\), \[\begin{align*} (x, y) \in f &\implies x \in \dom f \vee y = f(x) \\ &\implies x \in \dom g \vee y = g(x) \\ &\implies (x, y) \in g \end{align*}\] The other direction follows from the property2 of the equality \(f = g\).

Definition 2.8 A function \(f\) is

  • on \(A\) if \(\dom f = A\)
  • into \(B\) if \(\ran f \subset B\)
  • onto \(B\) if \(\ran f = B\)
Denote \(f: A \to B\) as a shorthand notation for \(f\) is a function on \(A\) and into \(B\).
Definition 2.9 (Identity) The identity function of a set \(A\) is defined as the function \[ \iota_A = \{ (x, y) : x,y \in A \vee x = y \} \] which exists as a susbet of \(A \times A\). Therefore, \((x, x) \in \iota_A\) iff \(x \in A\).
Definition 2.10 (restriction) The restriction of \(f\) to \(C\) is defined as the a subset of \(f\):
\[ f \restriction C = \{ (x, y) \in f: x \in C \} \] Immediately, \(f \restriction C = f \circ \iota_C\).
Proof. \[\begin{align*} (x, y) \in f \circ \iota_C &\iff \exists z, && x \iota_C z \vee zfy \\ &\iff \exists z, && z = x \vee x \iota_C z \vee zfy \\ &\iff &&x \iota_C x \vee xfy \\ &\iff &&x \in C \vee (x, y) \in f \\ &\iff &&(x, y) \in f \restriction C \end{align*}\]
Definition 2.11 (Injection) A function \(f\) is injective or one-to-one if for any \(x, y, z\), \(xfz, yfz\) implies \(x = y\). In other words, if \(f(x) = f(y)\) then \(x = y\) for all \(x, y \in \dom f\).
Definition 2.12 (Inverse) A function \(f\) is invertible if its inverse \(f^{-1}\) is also a function.
Theorem 2.3 A function \(f\) is invertible iff \(f\) is 1-1.
Proof. \[\begin{align*} &\forall x, y, z, && xfz \vee yfz \implies x = y \\ \iff &\forall x, y, z, && zf^{-1}x \vee zf^{-1}y \implies x = y \end{align*}\] by the definition of \(f^{-1}\).
