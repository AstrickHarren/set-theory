Axioms of Set Theory

Chapter 1 Axioms

Theorem 1.1 (Existence) There exists a set that contains no element, i.e.Â  \[ \exists A, \forall x, (x \notin A) \]
Definition 1.1 (Subset) If every element of set \(A\) is an element of set \(B\), then \(A\) is a subset of \(B\), denoted by \(A \subset B\): \[ \forall A, B, ((\forall x, x \in A \implies x \in B) \implies A \subset B) \]
Theorem 1.2 (Extensionality) Two sets are equal if they are subsets of each other. \[ \forall A, B, (A \subset B \vee B \subset A \implies A = B) \]
Definition 1.2 (Empty set) The set that contains no element is unique, i.e., \[ \forall A, B, ((\forall x, x \notin A) \vee (\forall x, x \notin B) \implies A = B) \]
Proof. \(\forall A, B\), \[\begin{align*} (\forall x, x \notin A) \vee (\forall x, x \notin B) &\implies \neg (\forall x, x \in A)\\ &\implies ((\forall x, x \in A) \implies (\forall x, x \in B)) \\ &\implies A \subset B \end{align*}\] and similarly \(B \subset A\), hence \(A = B\).
