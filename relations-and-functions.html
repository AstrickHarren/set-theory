Axioms of Set Theory

Chapter 2 Relations and Functions

Definition 2.1 (Ordered Pair) Define \((a, b) = \{\{a\}, \{a, b\}\}\).
Theorem 2.1 If \((a, b) = (x, y)\), iff \(a = x\) and \(b = y\).

Proof. If \(\{\{a\}, \{a, b\}\} = \{\{x\}, \{x, y\}\}\), and \(a \ne b, x \ne y\), then \(\{a\} \ne \{x, y\}\), otherwise \(x = y\), and likewise \(\{x\} \ne \{a, b\}\). Hence \(\{a\} = \{x\}\), and \(\{a, b\} = \{x, y\}\), showing \(a = x, b = y\) easily.

If \(a = b\), then \(\{a\} = (a, b) = \{\{x\}, \{x, y\}\}\), also showing \(b = a = x = y\) easily. Similar arguments concludes the case when \(x = y\).
Definition 2.2 Denote the class \[ \{ (x, y): P(x, y) \} = \{ z: \exists x, y, (z = (x, y) \vee P(z)) \} \] where \(P\) is any predicate, and \(P(x, y)\) is the shorthand notation of \(P((x, y))\).
Proposition 2.1 If \(A = \{ (x, y): P(x, y) \}\) is a class, then \[ (x, y) \in A \iff P(x, y) \] for all \(x, y\). Hence if for all \(x, y\), \(P(x, y)\) implies \((x, y) \in S\) for some set \(S\), then \(A\) exists as a subset of \(S\).

Proof. Observe \[\begin{align*} (x, y) \in A &\iff \exists x', y', &&(x, y) = (x', y') \vee P(x', y') \\ &\iff &&P(x, y) \\ \end{align*}\]

If \(P(x, y) \implies (x, y) \in S\), \[\begin{align*} z \in A &\implies \exists x, y, && z = (x, y) \vee P(x, y) \\ &\implies \exists x, y, && z = (x, y) \vee (x, y) \in S \\ &\implies && z \in S \end{align*}\]
Definition 2.3 (Cartesian Product) Define \[ A \times B = \{ (a, b): a \in A, b \in B \} \] which exists as a subset of \(\mathcal P (\mathcal P( A \cup B))\) because \((a, b) \subset \mathcal P(A \cup B)\) for all \(a \in A, b \in B\).
